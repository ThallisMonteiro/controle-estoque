# 📦 Sistema de Controle de Produtos

Projeto em desenvolvimento para controle de estoque com dashboard.

## 🚀 Funcionalidades

- Adicionar produtos
- Calcular preço total automaticamente
- Dashboard com quantidade de produtos
- Dashboard com valor total do estoque

## 🛠️ Tecnologias utilizadas

- HTML
- CSS
- JavaScript

## 📊 Dashboard

O dashboard mostra:

- Quantidade total de produtos
- Valor total em estoque
